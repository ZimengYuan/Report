## 技术架构解析

### Claude Code 核心特性

1. **Agent Teams** — Git Work Trees 并行工作，Context Bus 实时同步
2. **Programmable Hooks** — 拦截文件写入，强制代码规范检查
3. **交互式协作** — 关键架构决策点暂停请求确认

### Codex 核心特性

1. **Rust 构建** — 专为速度和 Token 效率设计
2. **自主沙箱** — 并行执行测试，brute force 解决方案
3. **agents.md 支持** — Linux Foundation 开放配置格式
